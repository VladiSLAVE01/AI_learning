Задание 2.1: Влияние размера ядра свертки
Эксперименты
Сравнивались 4 варианта:

Kernel 3x3

Kernel 5x5

Kernel 7x7

Mixed (1x1 + 3x3)

Результаты
Модель	Train Acc	Test Acc	Train Time
Kernel 3x3	99.7%	99.3%	93.0s
Kernel 5x5	99.7%	99.4%	85.1s
Kernel 7x7	99.7%	99.0%	92.6s
Mixed (1x1+3x3)	99.7%	99.0%	-
Выводы
Лучшая точность: Kernel 5x5 (99.4%)

Лучшее время: Kernel 5x5 (85.1s)

Kernel 3x3 показывает хороший баланс

Kernel 7x7 и Mixed уступают в точности

Задание 2.2: Влияние глубины CNN
Эксперименты
Сравнивались 4 варианта:

Shallow (2 conv слоя)

Medium (4 conv слоя)

Deep (6 conv слоев)

Deep + Residual

Результаты
Модель	Параметры	Точность	Особенности
Shallow (2 conv)	~457k	96.9%	Быстрое обучение
Medium (4 conv)	~789k	97.9%	Хороший баланс
Deep (6 conv)	~1.23M	98.1%	Высокая точность
Deep + Residual	~1.35M	99.0%	Лучшая точность
Выводы
Лучшая точность: Deep + Residual (99.0%)

Глубокие сети показывают лучшие результаты

Residual связи критичны для глубоких сетей

Batch Normalization помогает стабилизировать обучение

Общие выводы
Размер ядра
Оптимальный размер: 3x3 или 5x5

Большие ядра (7x7) дают больше контекста, но медленнее

Смешанные ядра эффективны, но сложнее в обучении

Глубина сети
Глубокие сети показывают лучшую точность

Требуется больше времени на обучение

Residual связи решают проблему затухания градиентов
