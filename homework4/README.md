Задание 3.1: Кастомные слои
Что сделали:
Мы создали несколько собственных слоев и сравнили их со стандартными:

1. Кастомный сверточный слой (CustomConv2d)

Добавляет шум к весам во время обучения (регуляризация)

Поддерживает dropout для целых каналов

Включает Batch Normalization

2. Attention механизм (CBAM)

Комбинирует канальное внимание (Squeeze-and-Excitation)

Пространственное внимание

Помогает модели фокусироваться на важных признаках

3. Кастомные функции активации

Swish: x * sigmoid(beta * x)

Mish: x * tanh(softplus(x))

Custom: обучаемая активация с параметрами

4. Кастомные pooling слои

AdaptiveMaxAvgPool: комбинация среднего и максимального pooling

FractionalMaxPool: дробный max pooling

StochasticPooling: вероятностный выбор элементов

Результаты:
Модель	Train Accuracy	Test Accuracy	Training Time
Standard CNN	99.4%	99.0%	76.4s
CNN with Custom Layers	99.5%	99.1%	86.5s
Выводы:
Кастомные слои дали небольшое улучшение (+0.1% точности)

Время обучения увеличилось на ~13% (из-за внимания и шума)

Inference время почти не изменилось

На MNIST разница минимальна, но на сложных данных эффект будет заметнее

Задание 3.2: Residual блоки
Что сделали:
Мы реализовали и сравнили различные Residual блоки:

1. Basic Residual Block

Два сверточных слоя с BatchNorm

Skip connection для передачи градиентов

Базовый вариант

Сложнее в обучении

Wide Residual - больше параметров, но стабильность ниже
