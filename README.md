# AI_learning
Здесь находятся выполненные домашние задания по Основам глубокого обучения и искусственного интеллекта

1. Тензоры (homework_tensors.py)
Тензоры - это просто многомерные таблицы с числами, как в Excel, только в Python.
Что научился делать:
Создавать тензоры разными способами:
случайные числа
нули или единицы
последовательности чисел (0, 1, 2, 3...)

Простые операции:
переворачивать матрицы (транспонирование)
умножать матрицы (как в школе на алгебре)
складывать все числа внутри тензора

Вырезать кусочки из тензоров:
брать отдельные строки или столбцы
брать кусок из середины
менять форму тензора (например, из 24 чисел сделать таблицу 4x6)

2. Автоматические градиенты (homework_autograd.py)

Задание 2.1
Взял числа x=2, y=3, z=4
Посчитал функцию f = x² + y² + z² + 2*x*y*z
Спросил у PyTorch "как изменится f, если чуть-чуть изменить x, y или z?"
PyTorch выдал правильные ответы, я проверил по формулам

Задание 2.2

Сделал простую модель: y = w*x + b
У нас были правильные ответы и мы считали ошибку (MSE)
PyTorch посчитал градиенты для w и b
Градиенты говорят нам: надо увеличить или уменьшить w и b, чтобы ошибка стала меньше

Задание 2.3

Посчитал производную функции f = sin(x² + 1) двумя способами:
автоматически (PyTorch)
вручную по формуле
Ответы совпали, значит PyTorch работает правильно

3. Производительность (homework_performance.py)
Проверил, насколько быстрее работает видеокарта (GPU) по сравнению с процессором (CPU).
Что делал:
Создавал большие матрицы
Считал сумму всех чисел в матрице
Замерял время на CPU и на GPU

Результат:

Для маленьких матриц разницы почти нет
Для больших матриц GPU работает в десятки раз быстрее
Поэтому нейросети обучают на видеокартах

Как запустить
Нужно установить PyTorch:

bash
pip install torch
Потом запустить любой файл:

bash
python homework_tensors.py
python homework_autograd.py
python homework_performance.py
Если есть видеокарта NVIDIA и установлен CUDA, то в третьем задании запустится GPU-тест.
